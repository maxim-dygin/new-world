# AGENTS.md

## Project

Новый Мир

## Mission

Build a peaceful, human-centered web project that explains the idea of a fairer planetary society: dignity, freedom of movement, equality of basic opportunity, fair labor, and global cooperation.

The project must feel calm, clear, humane, and accessible. It should not sound like propaganda, extremism, revenge, or a cult of personality.

## Core Principles

1. Peace over violence.
2. Human dignity over hierarchy.
3. Open dialogue over dogma.
4. Cooperation over domination.
5. Systems thinking over personal hatred.
6. Transparency over manipulation.
7. Practical steps over empty slogans.
8. No leader worship.

## Tone of Voice

Use language that is:

- calm;
- clear;
- emotionally honest;
- non-aggressive;
- anti-authoritarian but not hateful;
- understandable to ordinary people;
- hopeful without being naive.

Avoid:

- violent rhetoric;
- revenge language;
- calls to overthrow anyone by force;
- dehumanizing words;
- cult-of-personality framing;
- excessive ideological jargon;
- empty revolutionary aesthetics.

## Product Direction

The first website version should be a landing page with these sections:

1. Hero
2. The Problem
3. Problem Map
4. Manifesto
5. Principles
6. What Can I Do?
7. FAQ
8. Footer

## UX Rules

- Mobile-first.
- Simple navigation.
- Short text blocks.
- Clear headings.
- Soft but confident visual style.
- No aggressive red/black extremist aesthetic.
- No militaristic symbols.
- No leader portraits.
- No flags as the main identity.
- Use space, calm typography, and readable contrast.

## Suggested Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Component-based architecture
- Markdown documentation

## Code Style

- Keep components small.
- Use semantic HTML.
- Prefer readable names over clever abstractions.
- Avoid unnecessary dependencies.
- Keep copy editable and separated where reasonable.
- Make the project easy for a solo developer to maintain.

## Safety Boundaries

Do not create content that:

- encourages violence;
- instructs illegal activity;
- frames the project as a militant movement;
- targets ethnic, national, religious, or social groups;
- glorifies authoritarian leadership;
- encourages harassment;
- claims absolute truth.

When in doubt, rewrite toward peace, dignity, and lawful civic participation.

## First Implementation Goal

Create a clean working landing page that explains the project in a peaceful, UX-friendly way.

The first version does not need complex features. It should make the idea understandable in under 3 minutes.
